<section>

O elemento HTML <seção> representa uma seção autônoma genérica de um documento, que não tem um elemento semântico mais específico para representá-lo. As seções devem sempre ter um título, com pouquíssimas exceções. Exemplo:

Código:

            <section>
                <h2>Este é um título de seção</h2>
                <p>Este é um páragrafo de exemplo para esta seção</p>
            </section>
Resultado:

Este é um título de seção

Este é um páragrafo de exemplo para esta seção

<aside>

O elemento HTML <aside> representa uma parte de um documento cujo conteúdo está apenas indiretamente relacionado ao conteúdo principal do documento. Os elementos <aside> são frequentemente apresentados como barras laterais ou caixas de chamada. Desta forma, o que estiver em <aside> não precisa necessariamente ter relação com o assunto principal da página. Exemplos de conteúdo que podem ser incluídos em um <aside> incluem anúncios, classificados, links, etc.

É importante notar que a tag <aside> e a tag <section> têm propósitos diferentes e sua importância depende do contexto em que são usadas. A tag <section> é usada para agrupar conteúdo relacionado e geralmente inclui um título. A tag <aside>, por outro lado, é usada para conteúdo que pode ser considerado separado do conteúdo ao seu redor. Exemplo:

Código

            ...
            <p>O filme da Disney <cite>A Pequena Sereia</cite> foi lançado nos cinemas pela primeira vez em 1989.</p>
            <aside>
                <O filme arrecadou $87 milhões durante o primeiro lançamento.>
            </aside>
            <p>Mais informções sobre o filme...</p>
            ...
Resultado:

O filme da Disney A Pequena Sereia foi lançado nos cinemas pela primeira vez em 1989.

O filme arrecadou $87 milhões durante o primeiro lançamento.

Mais informções sobre o filme...

<nav>

O elemento HTML <nav> representa uma seção de uma página cuja finalidade é fornecer links de navegação, dentro do documento atual ou para outros documentos. Exemplos comuns de seções de navegação são menus, sumários e índices.

Para exemplificar nesta página foi feito um <nav> no <header> e pode ser visualizado no topo da página.


            <nav>
                <ul>
                    <li><a href="#">Item 1</a></li>
                    <li><a href="#">Item 2</a></li>
                    <li><a href="#">Item 3</a></li>
                </ul>
            </nav>
