Tags estruturais - <figure>, <figcaption> <picture>

<figure> e <figcaption>

O elemento HTML <figura> representa conteúdo independente, potencialmente com uma legenda opcional, que é especificada usando o elemento <figcaption> elementos. A figura, sua legenda e seu conteúdo são referenciados como uma única unidade.

O elemento HTML <figcaption> representa uma legenda ou legenda que descreve o restante do conteúdo de seu elemento pai <figura>.


            ...
            <figure>
                <img src="../img/gunfight.JPG" width="400" />
                <figcaption>BXL-Problema no Loader</figcaption>
            </figure>
                ...

<picture>

O elemento HTML <picture> contém zero ou mais elementos <source> e um elemento <img> para oferecer versões alternativas de uma imagem para diferentes cenários de exibição/dispositivo. O navegador considerará cada elemento filho e escolherá a melhor correspondência entre eles. Se nenhuma correspondência for encontrada — ou se o navegador não oferecer suporte ao elemento — a URL do atributo src do elemento será selecionada. A imagem selecionada é então apresentada no espaço ocupado pelo elemento.<source><picture><img><img>


            <picture>
                <source srcset="../img/marcando_paralela_small.jpg" media="(max-width:700px)">
                <img src="../img/marcando_paralela.jpg" />
            </picture>
